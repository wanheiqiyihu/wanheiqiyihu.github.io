4.正则表达式相关

当给你一大堆文本信息，让你提取其中的指定数据时，可以使用正则来实现。例如：提取文本中的邮箱和手机号

正则表达式并不是只属于python，在其他语言中同样存在。

4.1正则表达式

4.1.1字符相关

1.dhk 匹配文本中的dhk

2.[dhk]匹配d或h或k

3.匹配除了abc以外的字符串

4.[a-z]匹配a~z的任意字符，（[0-9]也可以）

5.”.”代指出换行符以外的任意字符

6.\w指代字母或数字或下划线（汉字）。

7.\d代指所有的数字

8.\s代指一的空白符，包括空格，制表符等。

2.+重复1次或更多次

3.?重复0次或1次

4.{n}重复n次

5.{n,}重复n次或多次

6.{n,m}重复n次到m次

python方法和函数拓展 sort sorted

1.sort 与 sorted的区别

sort是对列表本身进行永久性的排序。

sorted则会返回一个排序之后的列表，不会对列表本身进行修改。

sort 是应用在 list 上的方法，sorted 可以对所有可迭代的对象进行排序操作。

2.sort

2.1参数

  • cmp – 可选参数, 如果指定了该参数会使用该参数的方法进行排序。
  • key – 主要是用来进行比较的元素，只有一个参数，具体的函数的参数就是取自于可迭代对象中，指定可迭代对象中的一个元素来进行排序。
  • reverse – 排序规则，reverse = True 降序， reverse = False 升序（默认）。

2.2key参数的应用

根据列表中指定的元素进行排序

key会把列表的元素作为参数传入后面的函数，然后把函数的返回值作为排序的依据。

2.2.1扩展

如果key接受了两个返回值，那么第一个返回值就会是首要排序依据，如果第一个返回值相等，那么会根据第二个返回值进行排序。

2.3reverse参数的应用

reverse = True 降序， reverse = False 升序（默认）。
